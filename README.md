# TravelCheckinApp

I. Project Overview
TravelCheckinApp is a comprehensive travel management platform that integrates POS (Point of Sale) and PMS (Property Management System) modules. This project is designed as an ecosystem to manage hotels, restaurants, and various tourism services.

II. Key Features
1. POS and PMS Management: Includes specialized logic for handling room and floor management for accommodation services.
2. Smart Check-in System: Features a QR-based check-in mechanism for users at tourist locations.
3. SOS Emergency System: Integrated real-time emergency alerts and monitoring tools.
4. AI-Driven Analytics: Utilizes AI for travel itinerary planning and data management.

III. Technical Stack
1. Programming Languages: Primary use of TypeScript and PHP.
2. Frontend Development: React and Vite for the web-based administrative dashboard.
3. Mobile Development: React Native and Expo for the cross-platform mobile application.
4. Backend Infrastructure: Built using Node.js, Express, and PHP.
5. Database: MySQL and Firebase integration.

IV. Project Status
This project is currently a Work In Progress (WIP) and serves as part of the graduation thesis titled "SmartData AI-Flow". Development is active with a focus on smart data management and analytics.
